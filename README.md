# MyAUTH

一个专为 Andriod 手表打造的 TOTP 验证码生成器。

## Usage

1. 从 Release 页面下载 apk 并安装（例如通过 WearOS 工具箱）

2. 发送 配置文件 到手表的 内部存储/Android/data/cool.techo.auth/downloads 目录下，配置文件 为 每行一个的 otpauth 格式链接，部分验证器可直接导出

3. 启动程序
